# Patch Notes — Development Update

## Overview
This update introduces several core gameplay systems, major UI overhauls, and performance improvements. Key highlights include the addition of omnidirectional combat targeting, a full trade system, spellcasting mechanics, and extensive interface refinements.

---

## ✨ New Features

### Combat & Gameplay
- Added **omnidirectional ranged combat targeting system** for improved aiming flexibility
- Introduced **spell casting system** with **procedural spell effects**
- Enhanced overall **combat responsiveness and mechanics**
- Added **cinematic PK zone notifications**
- Implemented **death screen with respawn timer**

### Systems
- Implemented **Trade System** *(Credits: Calamity)*
- Added **SFX (Sound Effects) system**
