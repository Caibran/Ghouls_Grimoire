GitHub Publishing

Connect a GitHub Personal Access Token so every Save and Delete publishes directly to your repo — no extra steps.

1. Go to github.com/settings/tokens/new
2. Select repo scope only
3. Copy the token and paste it below
4. GitHub Pages rebuilds automatically (~30 sec)
